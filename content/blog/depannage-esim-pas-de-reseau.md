Vous venez d'atterrir, vous activez votre eSIM… et rien. Pas de réseau, pas d'internet. Pas de panique — dans la grande majorité des cas, le problème se règle en moins de 5 minutes avec une de ces 7 vérifications. Suivez-les dans l'ordre.

## 1. La data est-elle assignée à la bonne ligne ?

C'est de loin la cause la plus fréquente. Votre téléphone utilise peut-être encore votre SIM principale pour les données mobiles au lieu de l'eSIM.

**Sur iPhone** : Réglages → Données cellulaires → "Ligne eSIM" doit être sélectionnée
**Sur Android** : Réglages → Réseau → SIM → Données mobiles → choisir la ligne eSIM

Vérifiez ce point en premier — il résout le problème dans la moitié des cas.

## 2. Les données mobiles sont-elles activées ?

Simple mais souvent oublié, surtout si vous arriviez de l'avion avec le mode avion activé.

Descendez le menu rapide (centre de contrôle sur iPhone) et vérifiez que l'icône "Données mobiles" est bien activée. Vérifiez aussi dans Réglages → Données cellulaires que le bouton général est sur ON.

## 3. L'itinérance est-elle activée sur la ligne eSIM ?

Certains forfaits eSIM — notamment les forfaits régionaux multi-pays — nécessitent que l'itinérance soit activée pour fonctionner à l'étranger.

**Sur iPhone** : Réglages → Données cellulaires → votre ligne eSIM → activer "Données en itinérance"
**Sur Android** : Réglages → Réseau → SIM → eSIM → activer "Itinérance des données"

Attention : gardez l'itinérance **désactivée sur votre SIM principale** pour éviter tout risque de facturation surprise.

## 4. Mode avion + redémarrage

La technique classique qui fonctionne souvent quand les réglages semblent corrects mais que le réseau ne se connecte pas.

1. Activez le mode avion
2. Attendez 10 secondes
3. Désactivez le mode avion
4. Si ça ne suffit pas : éteignez complètement le téléphone, attendez 10 secondes, rallumez

Ce redémarrage force le téléphone à rechercher et s'enregistrer sur les antennes locales disponibles.

## 5. La sélection de réseau est-elle en automatique ?

Si vous avez déjà voyagé et sélectionné manuellement un opérateur dans un autre pays, ce réglage peut bloquer la connexion.

**Sur iPhone** : Réglages → Données cellulaires → Réseau → désactiver "Sélection manuelle"
**Sur Android** : Réglages → Réseau → SIM → Opérateur → Sélection automatique

Une fois repassé en automatique, le téléphone va chercher le meilleur réseau disponible localement — ça peut prendre 30 secondes.

## 6. L'APN est-il correctement configuré ?

Certains forfaits nécessitent la saisie manuelle d'un APN (Access Point Name). Si votre email de confirmation en mentionne un, vérifiez qu'il est bien renseigné.

**Sur iPhone** : Réglages → Données cellulaires → votre ligne eSIM → Configuration des données cellulaires
**Sur Android** : Réglages → Réseau → SIM → eSIM → Noms des points d'accès

Si aucun APN n'est mentionné dans votre email, cette étape ne s'applique pas à votre forfait.

## 7. Réinitialisation des paramètres réseau (dernier recours)

Si aucune des étapes précédentes n'a fonctionné, une réinitialisation des paramètres réseau règle presque tous les problèmes résiduels.

**Sur iPhone** : Réglages → Général → Transférer ou Réinitialiser → Réinitialiser → Réglages réseau
**Sur Android** : Réglages → Gestion générale → Réinitialiser → Réinitialiser les paramètres réseau

⚠️ **Note** : cette manipulation efface vos mots de passe Wi-Fi enregistrés. Notez-les avant si besoin.
