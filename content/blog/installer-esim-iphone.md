Votre QR code est arrivé par email. L'installation prend 3 minutes si vous suivez cette méthode dans l'ordre. Voici exactement ce qu'il faut faire — et ce qu'il ne faut pas rater.

## Avant de commencer : 3 vérifications rapides

**1. Votre iPhone est-il compatible eSIM ?**
Les iPhones compatibles eSIM commencent à partir de l'iPhone XS (2018). Si vous avez un modèle plus récent, vous êtes couvert. En cas de doute, vérifiez sur [notre page compatibilité →](/compatibilite)

**2. Votre iPhone est-il désimlockó ?**
Un iPhone verrouillé sur un opérateur ne peut pas utiliser une eSIM d'un autre opérateur. Vérifiez dans Réglages → Général → Informations → Verrouillage SIM. Si c'est indiqué "Aucun verrouillage SIM", vous êtes prêt.

**3. Êtes-vous connecté en Wi-Fi ?**
L'installation nécessite une connexion Wi-Fi stable. Faites-le chez vous ou dans un endroit avec un bon réseau — pas à l'aéroport au dernier moment.

---

## Installation : la méthode pas-à-pas

### Étape 1 — Ouvrez les réglages eSIM
Allez dans **Réglages → Données cellulaires → Ajouter une eSIM**

Sur les iPhones récents (iOS 17+), vous pouvez aussi aller dans **Réglages → Mobile → Ajouter une ligne**.

### Étape 2 — Scannez le QR code
Choisissez "Scanner le code QR" et pointez l'appareil photo vers le QR code reçu par email.

💡 **Astuce** : ouvrez l'email sur un ordinateur ou une tablette pour scanner plus facilement depuis votre iPhone. Si vous n'avez qu'un seul appareil, faites une capture d'écran du QR code et utilisez l'option "Entrer les détails manuellement" avec le code de confirmation.

### Étape 3 — Confirmez l'activation
L'iPhone vous demande de confirmer l'ajout de la nouvelle ligne. Appuyez sur **Activer** puis patientez 30 à 60 secondes.

### Étape 4 — Nommez votre ligne
Donnez un nom clair à votre eSIM — par exemple **"Voyage"** ou le nom du pays. Cela vous évitera toute confusion avec votre SIM principale.

### Étape 5 — Assignez les données mobiles à l'eSIM
C'est l'étape la plus importante et celle qu'on oublie le plus souvent.

Allez dans **Réglages → Données cellulaires** et vérifiez que la ligne sélectionnée pour les données est bien votre ligne **"Voyage"** (l'eSIM), pas votre SIM principale.

✅ Installation terminée. L'eSIM est prête — elle s'activera automatiquement à votre arrivée à destination.

## À l'arrivée : 2 minutes pour être connecté

Dès que vous atterrissez, désactivez le mode avion et vérifiez ces 3 points :

1. **Données cellulaires → Voyage** est bien sélectionné comme ligne data
2. La barre de signal affiche le réseau local (ex : "NTT Docomo" au Japon, "AT&T" aux États-Unis…)
3. Si le réseau ne s'accroche pas après 1 minute : activez l'**itinérance** sur la ligne eSIM uniquement (Réglages → Données cellulaires → Voyage → Données en itinérance → ON)

⚠️ **Important** : gardez l'itinérance **désactivée sur votre SIM principale** pour éviter tout frais de roaming de votre opérateur habituel.

## 3 réglages pour optimiser votre consommation

Une fois connecté, prenez 2 minutes pour faire ces réglages — ils peuvent vous économiser plusieurs gigaoctets sur un long séjour.

| Réglage | Où le trouver | Pourquoi |
|---|---|---|
| Désactiver les mises à jour automatiques | App Store → Mises à jour automatiques | Évite que vos apps se mettent à jour en data mobile |
| Vérifier la ligne data avant de quitter le Wi-Fi | Réglages → Données cellulaires | Vous évite de découvrir le problème dans la rue |
| Mode avion 10s puis OFF | Menu rapide | Relance l'enregistrement réseau si le signal est faible |
